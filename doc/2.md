Напишите программу, которая генерирует список из 100 случайных чисел в диапазоне от 1 до 100. Затем вычислите:

Среднее арифметическое,
Медиану,
Стандартное отклонение,
Округленное значение квадратного корня из суммы всех чисел.
Для реализации используйте функции и методы из библиотек random, math и statistics.

Пример работы (вывода):

`Среднее: 52.34, Медиана: 51.5, Стандартное отклонение: 28.91, Корень из суммы: 72.83`
