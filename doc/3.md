Напишите программу, которая генерирует случайный JSON-объект с данными о пользователе: `имя, возраст, email и пароль`.
Пароль должен быть случайной строкой из 12 символов (буквы, цифры, знаки препинания).
Сохраните этот JSON-объект в файл, а затем прочитайте его и выведите на экран.

Для реализации используйте функции и методы из библиотек random, json и `string`.
